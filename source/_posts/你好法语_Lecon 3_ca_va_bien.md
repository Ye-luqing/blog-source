# vocabulaire

bien
+ c'est bien.
+ Vous parlez très bien français.


aller

+ aller devant
+ aller en France
+ aller au cinéma
+ aller à Pékin

avoir
+  avoir deux enfants
+  Nous avons une voisine gentille.


# Grammaire

aller的现在时变位
+ je vais
+ tu vas
+ il va/elle va
+ nous allons
+ vous allez
+ ils vont/elles vont

+ Je vais en France.
+ Tu vas à Paris?
+ Il va au Japon.
+ Vous allez à Rome?

avoir的现在时变位
+ j'ai
+ tu as
+ il a/elle a
+ nous avons
+ vous avez
+ ils ont/elles ont
+ J'ai un secrétaire.
+ Tu as un café.
+ Il a une carte de visite.
+ Vous avez un rendez-vous

我的
 + mon+阳单
 + ma+阴单
 + mes+复数
 + mon secrétaire
 + ma secrétaire
 + mes secrétaire
 + C'est mon café.
 + Julie,c'est mon prénom.
